Финальный проект курса "Автоматизация тестирования с помощью Selenium и Python" (см. https://stepik.org/course/575).
В проекте представлены автотесты для проверки функциональности интернет-магазина. Код исполнен с применением паттерна Page Object.

НАЗНАЧЕНИЕ ФАЙЛОВ 

Файлы с тест-кейсами. Тесты для каждой web-страницы хранятся в отдельном файле.
test_main_page.py
test_product_page.py

Файл с локаторами. Локаторы элементов каждой web-страницы находятся в отдельном классе.
pages/locators.py

Базовая страница с классом BasePage, от которого унаследованы остальные классы. Здесь описаны вспомогательные методы и проверки, применимые к любой странице. 
pages/base_page.py 

Файлы страниц. В этих файлах объявлены классы и описаны методы для каждой web-страницы. Метод проверяет небольшую функциональность и затем вызывается в тест-кейсе.
pages/main_page.py
pages/login_page.py
pages/product_page.py
pages/basket_page.py
